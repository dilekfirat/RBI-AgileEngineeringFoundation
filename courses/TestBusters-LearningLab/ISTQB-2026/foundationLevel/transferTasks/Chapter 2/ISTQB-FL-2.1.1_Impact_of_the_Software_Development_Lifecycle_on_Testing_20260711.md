# How the SDLC Changes Testing on Toolshop

## 1. Title

## 2. Syllabus Reference

ISTQB FL – 2.1.1 Impact of the Software Development Lifecycle on Testing

## 3. Learning Objective

Explain in simple terms how testing differs when Toolshop is built in sprints (agile) instead of in one long sequential project.

## 4. Context / Scenario

Toolshop at [https://practicesoftwaretesting.com/](https://practicesoftwaretesting.com/) is developed in agile sprints. Each sprint delivers a working increment. Your task is to show how this changes testing compared to a traditional sequential project.
