# TBLL-ToolShop Project Assignment

## Project Title

**TBLL-ToolShop - Practical Software Testing Learning Journey**

## Introduction

Welcome to the TestBusters-LearningLab (TBLL).

This learning journey provides participants with the opportunity to apply software testing concepts and techniques in a realistic software project environment. The project is based on the open-source ToolShop application and is designed to support the learning objectives of several internationally recognized ISTQB® syllabi.

A key characteristic of this learning journey is the systematic use of Generative AI (GenAI) to support testing activities. Participants will learn how human testers and AI can collaborate effectively throughout the software testing lifecycle.

## Syllabi Covered

### ISTQB® Certified Tester Foundation Level (CTFL)

- Testing fundamentals
- Static testing
- Test analysis
- Test design techniques
- Test management concepts
- Tool support for testing

### ISTQB® Certified Tester Test Automation Engineer (CT-TAE)

- Test automation fundamentals
- Test automation architecture concepts
- Test automation design principles
- Test data strategies
- Automated test maintenance
- Test automation reporting

### ISTQB® Certified Tester Generative AI for Testing (CT-GenAI)

- Effective prompting techniques
- AI-assisted test design
- AI-assisted test data generation
- AI-assisted test analysis
- AI-assisted defect investigation
- Responsible use of Generative AI in testing

## GenAI-Supported Testing Approach

All test activities within the learning journey may be supported by Generative AI, including:

- Requirement analysis
- Test condition identification
- Test case generation
- Test charter creation
- Risk analysis
- Test data generation
- Defect report creation
- Test reporting
- Test automation support

Participants remain responsible for reviewing, validating, and approving all AI-generated outputs.

## Project Goals

1. Learn software testing by doing.
2. Apply ISTQB concepts in a realistic project.
3. Learn modern AI-assisted testing approaches.
4. Build a professional testing portfolio.
5. Gain confidence in testing activities.
6. Practice collaboration between human testers and AI tools.
