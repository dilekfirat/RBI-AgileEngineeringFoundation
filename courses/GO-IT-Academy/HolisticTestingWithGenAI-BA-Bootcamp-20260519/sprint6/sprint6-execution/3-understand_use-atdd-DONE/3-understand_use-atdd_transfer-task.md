
# Transfer Task: Test Design for Business Use Cases

## 1. Description

Apply T1T5 Test Design technique to design a set of T1T5 tests for business use cases.

---

## 2. Background

You are part of a product team preparing the next sprint.

To run acceptance testing for businss use cases that are influenced by the implemented user stories, 
you should define a set of T1T5 test to validate that the business use cases are working correct. 
For designing the test use the T1T5-Test-Design technique (3-understand_use-add/T1T5-testdesign-description.md)
