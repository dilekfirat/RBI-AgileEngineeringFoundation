# Black Story Rules

## Mandatory Structure

Every story MUST contain:
- Title
- Teaser
- Hidden Solution

---

## Teaser Rules

The teaser:
- must be short
- maximum 3 sentences
- each sentence starts with `-`
- must create curiosity immediately
- must hide critical information
- must not reveal the core twist

---

## Solution Rules

The solution:
- must fully explain the teaser
- must be realistic
- must create an "Aha!" effect
- must not rely on fantasy or magic
- maximum 3 sentences
- each sentence starts with `-`


---

## Difficulty Rules

Easy:
- few hidden assumptions

Medium:
- one major hidden assumption

Hard:
- multiple hidden assumptions

Brutal:
- requires perspective shift and systems thinking

---

## Forbidden Patterns

Never:
- create impossible scenarios
- use supernatural explanations
- create random endings
- rely purely on coincidence
- create stories without logical resolution
